# Support

Need help with the DLS AI Chatbot service? Here's where to go.

## Before You Ask

1. Check the [README](../README.md) — it covers setup, environment variables, the API contract, and the personalization system in detail.
2. Check existing [GitHub Issues](../../issues) to see if your question or bug has already been reported.
3. Confirm `JWT_SECRET` matches the main backend and `GROQ_API_KEY` is valid — these are the two most common sources of "it's not working."

## Getting Help

- **Bugs:** Open an issue using the [Bug Report template](./.github/ISSUE_TEMPLATE/bug_report.md).
- **Feature requests:** Open an issue using the [Feature Request template](./.github/ISSUE_TEMPLATE/feature_request.md).
- **Security issues:** Do **not** open a public issue — follow the process in [`SECURITY.md`](./SECURITY.md) instead.
- **General questions:** Open a discussion or issue with as much context as possible (Node version, whether you're hitting `/chat` or `/chat/stream`, relevant logs with secrets redacted).

## Response Times

This is a small, focused service maintained alongside the rest of the Digital Logics Studio / Boolforge ecosystem. There's no formal SLA, but maintainers try to triage new issues within a week.
